# Neural Network From Scratch in C++

A small C++ neural network project built from scratch for learning modern C++, numerical programming, and neural network internals.

This repository is intentionally written manually and contains **no AI-generated source code**.

## Status

This project is in an early experimental stage. It likely contains bugs, incomplete features, incorrect implementations, and possible security or stability issues.

Use at your own risk.

## Goals

* Learn modern C++ through practical implementation
* Build neural network components from scratch
* Focus on clean design, performance, and memory-aware code
* Avoid unnecessary abstractions and framework-like complexity
